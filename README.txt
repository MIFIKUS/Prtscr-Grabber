RUS
PhantomJS оставить в папке с программой
Для работы программы необходим Python 3.7, а так же установленные в нём библиотеки bs4, selenium, requests
Для установки этих библиотек необходимо запустить cmd от имени администратора и прописать следующие команды по очереди 
pip install bs4
pip install selenium
pip install requests

Когда вы выбираете куда сохранить скриншоты а конце указанного пути ставте \ 
D:\Projects не правильно           D:\Projects\ правильно
Так же если написать вместо пути default то будет выбран путь C:\PrtscrGrabber\

ENG
Leave PhantomJS in the program folder
The program requires Python 3.7, as well as libraries bs4, selenium, requests installed in it
To install these libraries, you must run cmd as an administrator and enter the following commands in turn
pip install bs4
pip install selenium
pip install requests

When you choose where to save the screenshots and the end of the path put \

D:\Projects False                  D:\Projects\ True
