PhantomJS оставить в папке с программой
Для работы  программы необходим Python 3.7, а так же установленные в нём библеотеки bs4, selenium, requests
Для установки этих библеотек необходимо запустить cmd от имени администратора и прописать следующие команды по очереди 
pip install bs4
pip install selenium
pip install requests
